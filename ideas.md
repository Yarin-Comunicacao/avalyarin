## Ideia 1: "Editorial Gastronômico" — Inspiração em revistas de alta gastronomia

**Design Movement:** Editorial Design / Magazine Layout
**Core Principles:** Tipografia como protagonista, grid assimétrico, fotografia em destaque, espaço negativo generoso.
**Color Philosophy:** Fundo off-white quente (#FAF7F2) com texto em preto carbono (#1A1A1A). Acentos em âmbar/dourado (#C8963E) para denotar qualidade e sofisticação acessível. Vermelho escuro (#8B2500) para CTAs e alertas.
**Layout Paradigm:** Grid editorial com colunas desiguais (2/3 + 1/3), cards que se sobrepõem levemente, seções com largura variável que quebram a monotonia.
**Signature Elements:** Linhas finas horizontais como separadores, números grandes estilizados para notas, ícones de linha fina (não preenchidos).
**Interaction Philosophy:** Transições suaves de página, cards que expandem com detalhes ao clicar, sliders para notas com feedback tátil.
**Animation:** Fade-in suave de elementos ao scroll, números de nota que contam de 0 até o valor final, cards que elevam com sombra ao hover.
**Typography System:** Display: "Playfair Display" (serif, bold) para títulos e notas grandes. Body: "Source Sans 3" (sans-serif) para texto corrido. Mono: para preços e dados numéricos.
</text>

## Ideia 2: "Neon Urbano" — A estética dos bares noturnos de São Paulo

**Design Movement:** Urban Neon / Dark UI com referências a letreiros de bar
**Core Principles:** Tema escuro como base, acentos vibrantes que simulam neon, atmosfera noturna, contraste alto.
**Color Philosophy:** Fundo escuro em camadas (#0D0D0D base, #1A1A1A cards, #252525 elevações). Neon âmbar (#FFB800) como cor primária (cerveja, luz quente). Neon rosa (#FF3366) para destaques. Branco suave (#E8E8E8) para texto.
**Layout Paradigm:** Layout vertical mobile-first com cards empilhados, navegação lateral fixa em desktop, seções que simulam "andares" de um prédio (cada categoria é um andar).
**Signature Elements:** Efeito glow sutil nos elementos interativos, bordas com gradiente neon, ícones que "acendem" ao hover, textura de concreto/tijolo como background sutil.
**Interaction Philosophy:** Elementos que "acendem" ao interagir, feedback visual imediato com brilho, transições que simulam luzes piscando.
**Animation:** Glow pulse nos botões ativos, slide-up para cards de estabelecimentos, contadores animados para notas, parallax sutil no hero.
**Typography System:** Display: "Bebas Neue" (condensada, impactante) para títulos — remete a letreiros. Body: "DM Sans" para legibilidade em fundo escuro. Números: "Oswald" para notas e preços.
</text>

## Ideia 3: "Caderno de Campo" — O guia pessoal do explorador gastronômico

**Design Movement:** Sketchbook / Field Notes / Handcrafted
**Core Principles:** Sensação de anotação manual, texturas de papel, elementos desenhados à mão, organicidade.
**Color Philosophy:** Fundo de papel envelhecido (#F5F0E8), texto em tinta escura (#2C2C2C), acentos em vermelho de caneta (#C0392B) para marcações e notas, azul de caneta (#2E4057) para links e ações.
**Layout Paradigm:** Layout de caderno com margens visíveis, "abas" laterais para categorias (como post-its), cards que parecem fichas coladas no caderno.
**Signature Elements:** Linhas de caderno como background, selos/carimbos para badges de avaliação, estrelas desenhadas à mão, bordas irregulares nos cards.
**Interaction Philosophy:** Arrastar e "colar" avaliações, checkboxes que parecem marcações de caneta, sliders que deixam "rastro" como um traço.
**Animation:** Elementos que "aparecem escritos" (typewriter effect), cards que "caem" como fichas no caderno, selos que "carimbam" ao completar avaliação.
**Typography System:** Display: "Caveat" (handwritten) para títulos e anotações. Body: "Lora" (serif elegante) para descrições. Mono: "Courier Prime" para dados e preços.
</text>
